# קבצי שאלות לדוגמה (לניתוח)

העלה לכאן את כל קבצי השאלות (DOCX, PDF, TXT) שברצונך שאנתח.

- **איך:** גרור קבצים לתיקייה הזו או העתק אותם לנתיב:
  `quizMDA/sample-questions/`
- **פורמטים נתמכים:** `.docx`, `.pdf`, `.txt`
- **חשוב:** קובץ **.doc** (Word ישן) לא נתמך — יש לשמור ב-Word כ־**שמור בשם → Word Document (.docx)**.
- **מטרה:** לפי המבנה של הקבצים אבנה/אשפר סקריפטים שינתחו קבצים נוספים בצורה מדויקת יותר.

אחרי שתשים כאן קבצים, כתוב ב-Chat: "הקבצים בתיקיית sample-questions מוכנים" ואנתח אותם.

---

**קבצים בתיקייה (לאחר ניתוח):**
- `מסכם קרדיו/מבחן מסכם קרדיולוגיה.doc` — מבחן קרדיולוגיה (~75 שאלות). הומר ל־`.txt` עם textutil לניתוח.
- `מסכם קרדיו/מבחן מסכם קרדיולוגיה.txt` — גרסת טקסט מלאה (747 שורות) — נותח; ה-parser מזהה 92 שאלות (כולל כותרת מסוננת).
- `מונחים ומושגים.doc` — מילון מונחים (לא שאלות). הומר ל־`מונחים ומושגים.txt`.
- `ANALYSIS.md` — דוח ניתוח מבנה הקבצים והמלצות לסקריפט.
